# Linux Tools

## 功能说明

这个脚本提供了一个交互式菜单系统，用于执行各种系统相关的任务和脚本安装。功能包括：

1. **系统相关**
   - 更新系统
   - 安装常用工具
   - 安装 Docker
   - 安装开发工具
   - 安装网络工具
   - 安装常用数据库
   - 安装 Node.js 和 npm
   - 清理不再需要的软件包
   - 更改系统名

2. **脚本大全**
   - 安装 kejilion 脚本
   - 安装 勇哥的SB 脚本
   - 安装宝塔开行版脚本
   - 还原到宝塔官方版脚本

## 使用方法

要运行此脚本并访问菜单，请执行以下命令：

bash <(curl -sSL https://raw.githubusercontent.com/vbskycn/linux-tools/main/linux.sh)
